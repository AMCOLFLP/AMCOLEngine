Printable Worksheets & Exams

Open Combined Assessments

Worksheet A: Equipment Identification

For each item, write the location, function, normal condition, and one possible defect: cylinder liner, turbocharger, centrifugal pump, generator breaker, bilge alarm.

Worksheet B: Instruction and Read-Back

Write a clear instruction and exact read-back for five operations: pump changeover, valve line-up, generator synchronization, filter cleaning, and emergency shutdown.

Written Examination

  1. Explain the function of five main-engine components.
  2. Describe a complete pump-starting sequence.
  3. Write a watch handover including one defect and one bypass.
  4. Write a maintenance report using passive voice.
  5. Write an initial emergency report for a machinery-space fire.

Speaking Examination

Student receives one scenario card and completes a three-minute operational exchange. Assess technical accuracy, grammar, pronunciation, closed-loop communication, and safety response.
