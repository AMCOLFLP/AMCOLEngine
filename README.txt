AMCOL Maritime English for Engineers — Interactive Learning Portal

START
1. Extract the complete ZIP file.
2. Keep every HTML file and the assets folder together.
3. Open login.html in a modern browser.

DEMO ACCOUNTS
- Administrator: username admin | password AMCOLadmin
- Teacher: username teacher | password AMCOLteacher
- Student: username student | password student1

COURSE UNITS
- Unit 1: Engine Room Familiarization & Safety
- Unit 2: Main Engine Parts & Functions
- Unit 3: Pumps, Piping, Valves & Fluid Systems
- Unit 4: Engine Room Watchkeeping & Logbook Communication
- Unit 5: Maintenance, Tools & Troubleshooting
- Unit 6: Electrical, Automation & Alarm Systems
- Unit 7: Emergencies, Pollution Prevention & Engine Department Coordination

FEATURES
- Same paginated AMCOL portal format as the supplied reference.
- Black, navy, and gold visual palette.
- Seven pages per unit: Overview, Vocabulary, Grammar, Technical Language, Conversation & Audio, Core Feature & Scenario, Practice & Check.
- Browser speech playback for model conversations and listening drills.
- Interactive vocabulary and multiple-choice checks.
- Student progress tracking and post-test unlock.
- Teacher/admin/student role simulation.
- Optional contextual Thai Assist.
- Printable pages and capstone simulation.

OFFLINE NOTE
This is a self-contained classroom prototype. Browser speech quality depends on the voices installed on the device. A public institutional deployment should use secure server-side authentication and a database.
